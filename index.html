Important aspects

Press the RED Select button to select the START Node. If you want to dismiss the current START Node click again and then, with the same color selected choose another cell.

Press the GREEN Select button to select the TARGET Node. If you want to dismiss the current TARGET Node click again and then, with the same color selected choose another cell.

Press the BLACK Select button to select the OBSTACLE Node. You are free to select as many OBSTACLE Nodes as you want.

Once the nodes have been selected, you can start the algorithm you want.
